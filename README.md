

This project is managed using uv.

To activate the environment, run "source .venv/bin/activate" from the project root (project folder).

To test everything has activated okay run "uv pip list".

### Testing

- to run unit-tests, run `uv run pytest` from the root project folder (runs all tests)

- to run a specific test file run `uv run pytest tests/test_file.py`
